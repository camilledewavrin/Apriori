README DE L'ALGORITHME APRIORI (DEWAVRIN | QUERO)

Dans un premier, il faut executer le script apriori.sql pour intégrer les données dans la base.
(ne pas oublier de créer une base appelée "apriori" sur mysql puis executer le script)

Ensuite, il faut lancer le site à partir du fichier "example.php" qui équivaut à l'index.php depuis
un serveur web du type WAMPServer.

L'algorithme se lance directement.
